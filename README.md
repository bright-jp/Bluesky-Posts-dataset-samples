この**Bluesky 投稿データセットサンプル**には**1,892件のレコード**が含まれており、**Bright Data API**を使用して抽出されました。

## データポイント

| フィールド名 | 説明 |
|------------|-------------|
| `url` | Blueskyプラットフォーム上の特定の投稿への直接URL。 |
| `post_id` | 各投稿の一意の識別子。 |
| `post_date` | 投稿が公開された日時。 |
| `posted_by` | ハンドル名やプロフィールリンクを含む、投稿を作成したユーザーに関する情報。 |
| `post_text` | 投稿のテキストコンテンツ。 |
| `comments` | 投稿に付けられたコメント数。 |
| `reposts` | 他のユーザーによって投稿が再投稿された回数。 |
| `likes` | 投稿が受け取った「いいね」の合計数。 |
| `reposted` | その投稿が他のユーザーのコンテンツの再投稿であるかどうかを示します。 |
| `external_links` | 投稿に含まれる外部Webサイトへのリンク。 |
| `images` | 投稿に添付された画像数。 |
| `videos` | 投稿に添付された動画数。 |
| `original_post` | コンテンツがオリジナルか再投稿かを示します。 |
| `pinned` | 投稿がユーザーのプロフィール上部に固定表示されているかどうかを示します。 |
| `discovery_input` | コンテンツ発見のために使用されるキーワードまたはタグ。 |
| `contains_video` | 投稿に動画が含まれているかどうかを示します。 |
| `profile_followers_count` | 投稿作成者のフォロワー数。 |
| `profile_follows_count` | 投稿作成者がフォローしているユーザー数。 |
| `profile_posts_count` | ユーザーによる総投稿数。 |
| `post_type` | テキスト、画像、動画などの投稿タイプ。 |
| `video_thumbnails` | 投稿に含まれる動画のサムネイル画像。 |
| `video_duration` | 投稿内の動画の長さ（秒）。 |


## ユースケース

### 1. ソーシャルメディア感情分析
投稿のテキストコンテンツとエンゲージメント指標を調査することで、さまざまな問題に対する公開センチメントを分析します。

### 2. インフルエンサー影響評価
フォロワー数とエンゲージメント率を分析して主要なインフルエンサーを特定し、さまざまなユーザーの影響力を評価します。

### 3. コンテンツパフォーマンスのベンチマーク
異なるプロフィール間で投稿のパフォーマンスを比較し、ベストプラクティスを特定してコンテンツ戦略を最適化します。
