## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Blueskyプラットフォーム上の特定の投稿への直接URLです。 |
| `post_id` | 各投稿の一意の識別子です。 |
| `post_date` | 投稿が公開された日時です。 |
| `posted_by` | ハンドルやプロフィールリンクを含む、投稿を作成したユーザーに関する情報です。 |
| `post_text` | 投稿のテキスト内容です。 |
| `comments` | 投稿に付いたコメント数です。 |
| `reposts` | 他のユーザーによって投稿がリポストされた回数です。 |
| `likes` | 投稿が受け取ったいいねの総数です。 |
| `reposted` | 投稿が他ユーザーのコンテンツのリポストであるかどうかを示します。 |
| `external_links` | 投稿に含まれる外部Webサイトへのリンクです。 |
| `images` | 投稿に添付された画像の数です。 |
| `videos` | 投稿に添付された動画の数です。 |
| `original_post` | コンテンツがオリジナルかリポストかを示します。 |
| `pinned` | 投稿がユーザーのプロフィール上部に固定表示されているかどうかを示します。 |
| `discovery_input` | コンテンツ発見のために使用されるキーワードまたはタグです。 |
| `contains_video` | 投稿に動画が含まれているかどうかを示します。 |
| `profile_followers_count` | 投稿作成者のフォロワー数です。 |
| `profile_follows_count` | 投稿作成者がフォローしているユーザー数です。 |
| `profile_posts_count` | ユーザーが作成した投稿の総数です。 |
| `post_type` | テキスト、画像、動画などの投稿タイプです。 |
| `video_thumbnails` | 投稿に含まれる動画のサムネイル画像です。 |
| `video_duration` | 投稿内の動画の長さ（秒）です。 |


## Use Cases

### 1. Social Media Sentiment Analysis
投稿のテキスト内容とエンゲージメント指標を調べることで、さまざまな問題に対する世論の感情を分析します。

### 2. Influencer Impact Assessment
フォロワー数とエンゲージメント比率を分析して、主要なインフルエンサーを特定し、さまざまなユーザーの影響力を評価します。

### 3. Content Performance Benchmarking
異なるプロフィール間で投稿のパフォーマンスを比較し、ベストプラクティスを特定してコンテンツ戦略を最適化します。
